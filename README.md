> # Kawa Resume
> 卡哇简历--开源程序员简历模板库（孵化中）

## 使用方法
### 安装依赖
```bash
pnpm install
```
### 启动项目
```bash
pnpm start
```
启动后访问 [http://localhost:5173](http://localhost:5173) 查看效果

## 技术栈

- React
- TailwindCSS
- JavaScript
- Vite
